Welcome to the Digital Audio Synth

An open source project that is free to distribute, modify and build upon.

This is a desktop application and will not function without a physical keyboard. You could use a tablet device with a keyboard, but mobile phones are not suitable and the interface will not scale down to mobile phone size screens. UPDATE OK! We decided that you should at least be able to view the site on mobile, so, we did that for you.

Have some fun, make some tunes for your games and let us hear your creations.

NOTE: If things get a little awry, amiss, as in the sound starts to distort and go "snap" "crackle" & "pop" just refresh the browser and everything will be back to normal.

CAUTION! The sounds are best heard through earphones / headphones but, TURN THE VOLUME DOWN FIRST!!! and ramp it up gradually or you will go deaf :-P

DA SYNTH DUDE

What you can do with this app:

New In Release 3:

Select music a file from your hard drive!

You can now select the music you have downloaded from the Media Recorder. Then play additional music / beats on top of that while recording at the same time.

Alternatively, load any music track from your computer and record on top of it!

  • play some musical notes
  • change the octave that the keyboard plays
  • control the decay (the duration that each note plays)
  • change the wave form
  • play some drums; kick, kettles, snare and Hi Hat
  • set the echo on the kettle drums for really funky sounds
  • record the music as you play
  • download the recording directly from the Media Recorder
  • upload the edited file (I use Audacity) as music.wav
  • play back the uploaded file and record on top of it
  • have some fun creating tunes

Note: to upload a file you will need an account to do so, just send me a message via the GitHub Repo and we'll set you up!
