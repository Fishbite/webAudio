Da Synth App

Welcome to the Digital Audio Synth

An open source project that is free to distribute, modify and build upon.

This is a desktop site and will not function without a physical keyboard. You could use a tablet device with a keyboard, but mobile phones are not suitable and the interface will not scale down to mobile phone size screens. There was no point making it do so.

But, have some fun, make some tunes for your games and let us hear your creations.

DA SYNTH DUDE

What you can do with this app:

  • play some musical notes
  • change the octave that the keyboard plays
  • control the decay (the duration that each note plays)
  • change the wave form
  • play some drums; kick, kettles, snare and Hi Hat
  • set the echo on the kettle drums for really funky sounds
  • record the music as you play
  • download the recording directly from the Media Recorder
  • upload the edited file (I use Audacity) as music.wav
  • play back the uploaded file and record on top of it
  • have some fun creating tunes

Note: to upload a file you will need an account to do so, just send me a message and we'll set you up!
